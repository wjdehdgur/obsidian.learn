## ✅ 셀 (Cell)

### 🔤 용어 해석  
- Cell: 세포, 칸, 격자  
- 컴퓨터에서는 표 형식 구조의 **데이터가 입력되는 최소 단위 공간**

---

### 📌 개념 정리  
셀(Cell)은 엑셀, 구글 시트, 스프레드시트 등에서 **데이터가 입력되는 기본 단위**로, **행(Row)**과 **열(Column)**이 교차하는 지점에 위치한다.  
각 셀은 고유한 주소(예: A1, B3 등)를 가지며, **값(텍스트, 숫자, 수식 등)을 저장하거나 계산 기능을 수행**할 수 있다.

셀은 하나하나 독립적이면서도 표 전체에서 상호작용할 수 있어, 데이터 구성, 정렬, 분석, 시각화의 핵심 기반이 된다.

---

### ⚙️ 구성 요소  
- **주소(Address)**: A1, B2 등 열+행 조합으로 구분  
- **내용(Content)**: 입력된 값 (숫자, 텍스트, 날짜, 수식 등)  
- **형식(Format)**: 텍스트 정렬, 색상, 테두리, 표시 형식 등  
- **참조 유형**: 상대참조(A1), 절대참조($A$1), 혼합참조(A$1 등)

---

### 🧭 흐름 / 방향성  
1. 시트가 생성되면 셀들이 격자 형태로 자동 생성  
2. 셀에 데이터를 직접 입력하거나 수식을 통해 연산  
3. 여러 셀을 연결하거나 참조하여 데이터 처리  
4. 범위 선택, 조건부 서식, 필터, 피벗 등 다양한 기능 활용 가능

---

### 💬 실무 예시  
- **재무제표 작성**: 각 셀에 수치 및 수식을 입력하여 자동 계산  
- **데이터 집계**: 셀 범위를 지정하여 SUM, AVERAGE 등 함수 적용  
- **조건부 서식**: 특정 셀 값에 따라 색상 자동 변경  
- **함수 참조**: =B2+C2, =IF(A1>10,"Y","N") 등 수식 계산

---

### 🔁 관련 개념 비교  
| 개념 | 셀(Cell) | 필드(Field) |
|------|----------|-------------|
| 위치 | 시트 내 좌표로 지정 (A1 등) | DB 테이블 내 열 이름으로 지정 |
| 데이터 단위 | 하나의 값 | 하나의 속성 |
| 관계 | 독립적이면서 상호 참조 가능 | 튜플 단위로 묶여 사용 |
| 사용 환경 | 엑셀, 스프레드시트 등 | 관계형 데이터베이스 |

---

### 🎯 핵심 요약  
- 셀은 표 형식 문서에서 데이터를 입력하거나 계산하는 최소 단위  
- 주소 지정과 참조 기능을 통해 강력한 계산/분석 가능  
- 실무 문서 처리의 기본 구성 요소

---

### 🧠 학습 메모  
- 절대 vs 상대 참조 차이는 실무에서도 자주 혼동되므로 꼭 연습  
- VBA, 스크립트 등에서 셀 객체(Cell Object)로 제어 가능  
- 셀 병합(Merge)과 서식 조정은 시각적 가독성에도 중요
